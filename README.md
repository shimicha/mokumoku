## 事業をエンジニアリングしよう提案編の回答は以下に記述してください
<選択した事業側の課題>
サービス登録者数の内、男性60%に対して、女性は40%。一方で、サービス内のもくもく会に参加した人の比率は、男性90%：女性10%と大きな差が出ています。もっと女性が使いやすいようなサービス設計にする必要があるのではないか？

<提案内容>
現状は相談できる女性メンターと繋がる機能が無いため、女性が会に参加する心理的ハードルが高くなっていることが考えられる。サービス内で経験豊富な女性メンターを配置し、マイページ画面等に「女性メンターと繋がる」ボタンを設置して女性メンターとセッションできるようにし、女性ユーザーの不安払拭や心理的ハードルを下げるサービスを提供する。

<実装方針>
・「女性メンターと繋がる」ボタンを設置したと仮定。ボタンを押すとメンター一覧ページに遷移。検索フォーム設置して、メンターの専門分野等をタグ条件にして自分に合ったメンターが探せるようにする。
・セッション内容をユーザーに返すフィードバック機能を提供する。
・カレンダーにセッション実施済のタグを設置して、そこをタップするとセッション内容を確認できるページに遷移させる。
